Quick Start

With the CWE Workflow tool, a TACC account, an internet connection, and a geometry file, you can easily generate a mesh and run a wind flow simulation. For an extra-short quick start guide, check the tool's Workflow Diagram.

Create a Case

Each case is stored in a remote folder within your account on DesignSafe-CI. You can create a new case and then upload a geometry or duplicate a case from an existing one.

Generate a Mesh

Select your case and click 'Parameters' → 'Mesh Parameters'. Select one of the default geometries or a previously uploaded geometry file. You can set the mesh density, subdomains, and boundaries. Save your parameters using the 'save all parameters' button. Save all parameters, then press 'run' to submit the mesh generation process on the remote server.

Running a Simulation

Once the mesh is generated, go to the 'Parameters' tab and then the 'Simulation Parameters' tab. Adjust simulation duration, time step, inflow velocity, kinematic viscosity and other parameters. Again, press 'save all parameters' before clicking 'run' to submit the simulation to TACC's remote server.

Jobs

Once a mesh generation or simulation job has been submitted, you'll see it running and can get status updates under the 'Remote Jobs' tab.

Results

Once a job is finished, select a case to view the mesh or simulation results. All case files can be easily downloaded from the 'View Results' tab, using the 'Download Entire Case' button.
