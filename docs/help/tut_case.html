Step 1: Create a Case

There are multiple ways to create a new CFD Simulation case. With the CWE Tool you may

  1. Create a NEW case
  2. CLONE an existing case

Create a NEW case

  • Click on the tab
  • Click on the Button
  • Select a folder to contain the new case [1]
  • Type a name for the new case [2].
  • Select a template for the type of case [3]. We recommend 2D simulation for your first case.

After a short wait, the new case will be available for you to run. You can select it in the 'Simulation Cases' tab.

Continue to Step 2: Create a Mesh

CLONE an existing case

  • Cloning a case operates similar to creating a case. However, use the 'Duplicate Case' button instead of 'Create New Case'.
